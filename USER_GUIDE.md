# FC 视频文字排版工具 — 使用说明

## 功能概览

FC 是一款专为商家和内容创作者设计的视频文字排版工具。  
内置三类精选预设样式，拖拽即可定位文字，一键合成导出。

---

## 使用流程（四步完成）

```
① 导入视频  →  ② 选择模板  →  ③ 编辑 & 拖拽  →  ④ 导出分享
```

---

## 详细操作说明

### 第一步：导入视频

打开 App 进入首页，有两种方式：

| 方式 | 操作 |
|------|------|
| 从相册导入 | 点击「**从相册选择视频**」，在系统相册中选择视频文件 |
| 实时拍摄 | 点击「**拍摄新视频**」，允许相机权限后直接录制 |

> 支持 MP4、MOV 等常见视频格式；建议时长控制在 3 分钟以内。

---

### 第二步：选择预设模板

进入编辑界面后，顶部展示三种模板卡片：

#### 🛒 电商爆款
适合淘宝、拼多多、抖音小店等电商场景

| 字段 | 默认内容 | 说明 |
|------|---------|------|
| 商品名称 | 爆款新品 | 左上角大字 |
| 现价 | ¥99 | 红色大号价格 |
| 原价 | 原价¥199 | 灰色小字，对比用 |
| 折扣标签 | 限时5折 | 黄色背景标签 |
| 立即抢购 | 立即抢购 >> | 橙色 CTA 按钮样式 |

#### 🏪 实体门店
适合餐饮、零售、服务业等线下门店宣传

| 字段 | 默认内容 | 说明 |
|------|---------|------|
| 店铺名称 | XX 旗舰店 | 居中顶部大字 |
| 店铺口号 | 品质保证 · 诚信经营 | 金色小字 |
| 地址 | 📍 XX市XX区XX路88号 | 左下角 |
| 电话 | 📞 400-000-0000 | 左下角 |
| 营业时间 | 🕙 09:00 – 22:00 | 左下角 |

#### 🎉 活动促销
适合节日促销、会员日、品牌活动等场景

| 字段 | 默认内容 | 说明 |
|------|---------|------|
| 活动主题 | 年中大促销 | 金色居中大字 |
| 副标题 | 全场低至 1 折起 | 白色居中 |

点击模板卡片，文字字段自动填充到视频预览上。

---

### 第三步：编辑与排版

#### 3.1 拖拽调整位置
- 在视频预览区域，**长按并拖动**文字即可任意移动位置
- 被选中的文字显示**蓝色边框**，拖动时显示**黄色边框**
- 点击空白区域取消选中

#### 3.2 点击展开编辑面板
- 在下方字段列表中点击任意字段，展开详细编辑面板
- 可修改：**文字内容 / 字号（10–80）/ 颜色 / 加粗**
- 点击字段左侧的 👁 图标可**显示或隐藏**该字段

#### 3.3 颜色选择
编辑面板提供 8 种快捷颜色：白、黑、红、橙、黄、绿、蓝、紫

---

### 第四步：导出与分享

1. 编辑完成后，点击右上角「**导出**」按钮
2. App 将使用 FFmpeg 把文字合成到视频中
3. 合成完成后，视频自动保存到：
   - **相册 → Movies → FCVideo** 文件夹
4. 点击「**分享视频**」可直接发送到微信、微博、抖音等平台
5. 点击「**重新制作**」回到首页开始新项目

---

## 权限说明

| 权限 | 用途 |
|------|------|
| 相机 | 拍摄视频 |
| 读取媒体（视频） | 从相册选择视频 |
| 写入存储（Android 9及以下） | 保存合成视频 |

---

## 注意事项

- 合成耗时取决于视频时长和设备性能，请耐心等待
- 请勿在合成过程中切换到后台或锁屏（部分设备可能中断）
- 导出的视频为 H.264 编码 MP4 格式，兼容所有主流平台
- 如遇导出失败，可返回编辑页面重试
